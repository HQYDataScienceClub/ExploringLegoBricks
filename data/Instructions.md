# Download Instructions

Download the `colors.csv` and `sets.csv` files from https://rebrickable.com/downloads
and save them to the directory: ExploringLegoBricks/data/

```
ExploringLegoBricks
    ├─ data
    │   ├── Instructions.md
    │   ├── colors.csv
    │   └── sets.csv
    ├─ lib
    ├─ src
    └── README.md
```
